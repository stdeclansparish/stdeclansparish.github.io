Preparing Mass Slideshows for the Weekend

Each week, six PowerPoint slideshows are prepared for the weekend Masses. These are based on seasonal templates, customized using the Elvanto service records, and saved in a structured archive.

Tools & Resources
  • Google Drive for Desktop
    • Optional, but highly recommended.
    • Makes it much quicker and easier to update files.
  • PowerPoint
  • Elvanto
    • Service plans and song lyrics for new songs are found here
  • Template files
    • in Mass Slides/Slideshow Templates/<Liturgical Season>/
  • Lyrics files:
    • Mass Slides/Overhead Lyrics.ppt
    • Mass Slides/Live Stream Lyrics.ppt

Step-by-Step Instructions

1. Create the Google Drive Folder Shortcut

  • Use the Elvanto service record to determine the liturgical date (e.g. 15th Sunday in Ordinary Time, Year B).
  • Navigate to the corresponding [Liturgical Date]/ folder in Mass Slides/Archives/.
  • Add a shortcut to that folder inside Mass Slides/Current Mass Slides/.
    You must use 'Add shortcut to Google Drive' and not 'Create Shortcut' or the shortcut will not work correctly on all devices.
  • Name it using the calendar date of the Sunday (e.g. 2025-07-13).

2. Create Slideshows from Templates

  • Navigate to Mass Slides/Slideshow Templates/[Season]/ and double click on the appropriate templates to create new .ppt files from them.
  • Save each file over its corresponding file in the archived folder you linked in step 1.
    You will need to create multiple new files from the Chanted Ordinary template, as it is used for 9am, 11am, and 5pm.
Filename Template Notes
vigil.ppt Overheads Template.potx Does not include chanted responses
7am.ppt No Music Overheads Template.potx No hymns; includes texts for entrance and communion chants
9am.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
11am.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
5pm.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
LiveStream.ppt Live Stream Template.potx Green background, text at bottom, to allow layering with video feed

3. Fill in Slides Using Elvanto & Lyrics Files

  • Use the Elvanto service record to identify and paste in:
    • Psalm response and verses (same for all Masses)
    • Alleluia verse (same for all Masses)
    • Entrance and Communion Antiphons (7am only)
  • Open the appropriate lyrics file and copy the section for each song:
    • Includes title slide, lyrics slides, and a blank slide at the end.
    • Hymns not needed for 7am.
  • If a hymn is missing:
    • Check Elvanto for copyright info (CCLI, One License, or Public Domain).
    • Add a new section to the lyrics file using the correct title slide layout.

4. Special Notes for Livestream

  • Use the Live Stream Template as-is.
  • It includes a green background and bottom-aligned text box with transparency.
  • No manual formatting is needed—just insert the correct content.
Final Checks
  • All slideshows are saved in the correct archive folder.
  • The shortcut in Current Mass Slides is correctly named and linked.
  • Lyrics are complete and properly formatted.
  • Livestream text appears correctly at the bottom of the screen.
It is recommended that you install Google Drive for Desktop, as it makes this process much easier. This version of the guide is for those who can't use that version of Google Drive.
If you are using Google Drive in the browser, you will need to download any file you are using, whether you are editing it or simply copying from it. You will also need to re-upload any files you edit once you are done.

1. Create the Google Drive Folder Shortcut

  • Use the Elvanto service record to determine the liturgical date (e.g. 15th Sunday in Ordinary Time, Year B).
  • Navigate to the corresponding archive folder.
  • Add a shortcut to the Google Drive folder Mass Slides/Current Mass Slides/ of the archive folder.
    • Right-click the folder, click its actions button (⋮) or select the folder and hit Alt + A
    • Select Organise > Add Shortcut
    • Choose Mass Slides/Current Mass Slides/ to place the shortcut there.
  • Name it using the calendar date of the Sunday (e.g. 2025-07-13).

2. Download Templates and Lyrics Libraries

If you have just finished doing another set of slides you can skip this step, but it is best to have a recent copy, as both the template files and the library files can change from time to time.
  • Navigate to Mass Slides/Slideshow Templates/[Season]/ and download the template (.potx) files.
  • Navigate to Mass Slides/ and download Overhead Lyrics.pptx and Live Stream Lyrics.pptx.

3. Create Slideshows from Templates

  • Double-click on the template files downloaded in step 2 to create a new slideshow for each of the Mass times, saving them as outlined in the table below.
    You will need to create multiple new files from the Chanted Ordinary template, as it is used for 9am, 11am, and 5pm.
Filename Template Notes
vigil.ppt Overheads Template.potx Does not include chanted responses
7am.ppt No Music Overheads Template.potx No hymns; includes texts for entrance and communion chants
9am.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
11am.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
5pm.ppt Overheads Template - Chanted Ordinary.potx Includes chanted responses
LiveStream.ppt Live Stream Template.potx Green background, text at bottom, to allow layering with video feed

4. Fill in Slides Using Elvanto & Lyrics Library Files

  • Use the Elvanto service record to identify and paste in:
    • Psalm response and verses (same for all Masses)
    • Alleluia verse (same for all Masses)
    • Entrance and Communion Antiphons (7am only)
  • Open the appropriate lyrics library file (downloaded in step 2) and copy the section for each song:
    It is easiest if you select a song's lyrics by clicking on its section title in the panel on the left. This selects all the slides for the song, including its title slide and empty final slide, as well as its section title.
    • Includes title slide, lyrics slides, and a blank slide at the end.
    • Hymns not needed for 7am.
  • If a hymn is missing:
    • Check Elvanto for copyright info (CCLI, One License, or Public Domain).
    • Add a new section to the lyrics file using the correct title slide layout.
  • Upload the new files to the shortcutted Archive folder, overwriting the old ones.

  • No manual formatting is needed—just insert the correct content.
Final Checks
