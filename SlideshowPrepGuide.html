Preparing Mass Slideshows for the Weekend

A step-by-step guide for volunteers and staff

Overview (Desktop Instructions)

Each week, six PowerPoint slideshows are prepared for the weekend Masses. These are based on seasonal templates, customized using the Elvanto service records, and saved in a structured archive.

Tools & Resources
  • Google Drive for Desktop
  • PowerPoint
  • Elvanto (for service records and song lyrics)
  • Template files in Mass Slides/Slideshow Templates/<Liturgical Season>/
  • Lyrics files:
    • Mass Slides/Overhead Lyrics.ppt
    • Mass Slides/Live Stream Lyrics.ppt
Folder Structure
Folder Path Purpose
Mass Slides/Archives/<Season>/Yr <A|B|C>/<Liturgical Title>/ Where slideshows are saved
Mass Slides/Current Mass Slides/ Shortcut to current liturgical folder (named by calendar date)
Mass Slides/Slideshow Templates/<Season>/ PowerPoint templates
Mass Slides/Overhead Lyrics.ppt Lyrics for Vigil, 9am, 11am, 5pm
Mass Slides/Live Stream Lyrics.ppt Lyrics for livestream (9am)
Step-by-Step Instructions

1. Identify the Liturgical Date

  • Use the Elvanto service record to determine the liturgical title (e.g. 15th Sunday in Ordinary Time, Year B).
  • Navigate to the corresponding archive folder.

2. Create a Shortcut

  • Add a shortcut to the Google Drive folder Mass Slides/Current Mass Slides/ of the archive folder.
    • You must use 'Add shortcut to Google Drive' and not 'Create Shortcut' or the shortcut will not work correctly on all devices.
  • Name it using the calendar date of the Sunday (e.g. 2025-07-13).

3. Create Slideshows from Templates

Mass Template Filename Notes
Vigil Overheads Template vigil.ppt Standard music template
7am No Music Overheads Template 7am.ppt No hymns; includes chant texts
9am Overheads Template 9am.ppt Standard music template
11am Overheads Template With Plainchant 11am.ppt Includes chanted responses
5pm Overheads Template 5pm.ppt Saved in Overheads/ subfolder
Livestream Live Stream Template LiveStream.ppt Green background, text at bottom

To create a new file:

  • With Google Drive for Desktop: Double-click the .potx file to create a new .ppt file.
  • Without it: Download the template, double-click it locally, then upload the new file to the archive folder.

4. Fill in Slides Using Elvanto & Lyrics Files

  • Use the Elvanto service record to find:
    • Psalm (same for all Masses)
    • Alleluia verse (same for all Masses)
    • Hymns (except 7am)
  • For 7am: insert Entrance and Communion chants instead of hymns.
  • Open the appropriate lyrics file and copy the section for each song:
    • Includes title slide, lyrics slides, and a blank slide at the end.
  • If a hymn is missing:
    • Check Elvanto for copyright info (CCLI, One License, or Public Domain).
    • Add a new section to the lyrics file using the correct title slide layout.

5. Special Notes for Livestream

  • Use the Live Stream Template as-is.
  • It includes a green background and bottom-aligned text box with transparency.
  • No manual formatting is neededâ€”just insert the correct content.
Final Checks
  • All slideshows are saved in the correct archive folder.
  • The shortcut in Current Mass Slides is correctly named and linked.
  • Lyrics are complete and properly formatted.
  • Livestream text appears correctly at the bottom of the screen.
